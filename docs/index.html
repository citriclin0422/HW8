SVM Kernel Trick 3D Demo

An interactive learning project that connects maximum margin intuition, support vectors, nonlinear kernels, and rotatable 3D Plotly visualizations.

Deploy on Streamlit GitHub Repository

Three Phase Learning Flow

Phase 1: Manim Concept

Show nonlinear 2D data, margin intuition, kernel mapping, and a 3D separating plane.

Phase 2: sklearn Model

Generate datasets, train Linear and RBF SVMs, evaluate metrics, and highlight support vectors.

Phase 3: Streamlit + Plotly

Control C, gamma, dataset shape, and noise while exploring live 2D and 3D decision plots.
